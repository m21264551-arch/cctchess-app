# CCT Chess Screenshots

Generated screenshots for the first App Store Connect listing.

Apple accepts PNG screenshots and requires iPad screenshots when an app runs on iPad. These assets cover the primary iPhone and iPad device families for this universal app.

## Files

| Device family | Display class | File | Size |
| --- | --- | --- | --- |
| iPhone | 6.9 inch portrait | `Screenshots/iPhone-6.9/01-home.png` | 1320 x 2868 |
| iPad | 13 inch portrait | `Screenshots/iPad-13/01-home.png` | 2064 x 2752 |

## Capture Notes

- Captured from the real CCT Chess app running in iOS Simulator.
- Status bar was normalized to 9:41, full Wi-Fi, and full battery.
- Screenshots show the first-run home screen with no user data.
